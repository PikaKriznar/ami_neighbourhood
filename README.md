# ami_neighbourhood
Interaktivni vizualizator prikazuje energetsko skupnost več hiš z različno porabo in proizvodnjo. Uporabnik spreminja parametre in opazuje tokove energije, stroške ter vpliv na okolje. Sistem spodbuja razumevanje medsebojne odvisnosti, primerjavo scenarijev in refleksijo o rabi energije skozi preprosto, pregledno in interaktivno predstavitev.
